	访问:/system/appversion?_method=get_version&platform=
	(GET)
	返回:error_code,version,新版本连接

首页：
获取首页轮播图
	访问:/index/indexbanner?_method=get
	(GET)
	返回:error_code,轮播图列表
获取同城前10条热门老师/10最新需求
	访问:/product?_method=get_product
	(POST) param: city,status,page
	返回一页热门老师/最新需求列表
